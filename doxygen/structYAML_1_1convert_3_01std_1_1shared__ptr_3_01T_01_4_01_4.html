Detailed Description

template<class T>
struct YAML::convert< std::shared_ptr< T > >

Specializatio of YAML::convert for std::shared_ptr, this is used to redirect the serialization of a derived class to the specialization of the serialization for a base class, for example all subclasses of Component can use the Component serialization specialization, currently each redirection has to be implemented separately, there is probably a way to do this automatically.
