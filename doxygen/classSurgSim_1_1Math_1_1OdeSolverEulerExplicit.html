Detailed Description

Euler Explicit ode solver.

Note
M(x(t), v(t)).a(t) = f(t, x(t), v(t))
This ode equation is solved as an ode of order 1 by defining the state vector y = (x v)^t:
y' = ( x' ) = ( dx/dt ) = ( v )
( v' ) = ( dv/dt ) = ( M(x, v)^{-1}.f(x, v) )
y' = f(t, y)
Euler Explicit is also called forward Euler as it solves this integral using a forward evaluation:
y' = (y(t+dt) - y(t)) / dt
which leads to the integration scheme:
{ x(t+dt) = x(t) + dt.v(t)
{ v(t+dt) = v(t) + dt.a(t)
