Detailed Description

Generate a constraint for every contact that was calculated.

The general algorithm is such, for each pair of Collision Representations that has Contacts For each contact:

  • Generate a localization for each side of the pair
  • Find the correct CollisionImplementation for the Representation and type of constraint, note that currently we use only one type of constraint for the collision
  • Create the ConstraintData for the contact with
  • Create a constraint from, the ContactImplmentation, Localization and ConstraintData
  • Add it to the list of Constraints At the end those constraints are added as contact constraints to the physics state
